```git
.project

.classpath

.idea/

*.ipr

*.iml

*.iws

*.log

*.cache

*.diff

*.patch

*.tmp



# system ignore

.DS_Store

Thumbs.db



# package ignore (optional)

*.jar

*.war

*.zip

*.tar

*.tar.gz
```
